# Contributing to Audio Enhancer Pro

First off, thank you for considering contributing to Audio Enhancer Pro! It's people like you that make this tool great.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally: `git clone https://github.com/your-username/volume-booster.git`
3.  **Create a new branch** for your feature or bugfix: `git checkout -b feature/your-feature-name`
4.  **Make your changes** in the codebase.
5.  **Test your changes** thoroughly in Brave/Chrome by loading the extension locally (see README instructions).
6.  **Commit your changes** with a clear and descriptive commit message.
7.  **Push to your fork**: `git push origin feature/your-feature-name`
8.  **Submit a Pull Request** to the main repository.

## Code Style

- Write clean, well-documented JavaScript (Vanilla).
- Ensure any UI changes are responsive and accessible.
- Maintain existing code structure and naming conventions.

## Bug Reports and Feature Requests

If you encounter a bug or have a suggestion, please open an issue in the GitHub repository. Provide as much detail as possible, including browser version, steps to reproduce, and screenshots if applicable.

Thank you for your support!
